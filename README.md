# RestfulAPI-Example
An example to demo how to work with restful API

## Build Setting
* iOS 10.2+
* Xcode 8.2.1+
* Swift 3.0+

## Requirement
* Carthage

## Dependencies
* Alamofire
* SwiftyJSON
* KeychainAccess

```sh
github "SwiftyJSON/SwiftyJSON" ~> 3.1
github "Alamofire/Alamofire" ~> 4.4
github "kishikawakatsumi/KeychainAccess" ~> 3.0
```

## Installation
```sh
$ carthage update --platform ios
```
