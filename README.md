# RestfulAPI-Example
An example to demo how to work with restful API

## Build Setting
* iOS 10.2+
* Xcode 8.2.1+
* Swift 3.0+

## Frameworks
* Alamofire
* SwiftyJSON

```
github "SwiftyJSON/SwiftyJSON" ~> 3.1
github "Alamofire/Alamofire" ~> 4.4
```

## Installation
```bash
$ carthage update --platform ios
```
